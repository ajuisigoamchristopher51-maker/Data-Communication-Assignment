Section 01

Introduction to
Data Communication

Data communication is the exchange of data between two or more devices through a transmission medium such as a wire cable. It is the foundation of modern computing and networking, enabling everything from simple text messages to live video calls.

Five Components

Every data communication system has five core components: the Message, Sender, Receiver, Transmission Medium, and Protocol. Without all five, communication breaks down.

Modes of Transmission

Simplex — one-way only (TV broadcast). Half-duplex — two-way, but not simultaneously (walkie-talkie). Full-duplex — simultaneous two-way (phone call).

A Brief History

Data communication began with Samuel Morse's telegraph in 1837, evolved through telephone networks, radio, and fiber optics, and culminated in today's global internet — the world's largest data communication network.

Section 02

Transmission Media
& Signals

The quality, speed, and range of data communication depends heavily on the type of transmission medium and the nature of the signal used to carry information.

Analog Signal vs Digital Signal | Guided Media Unguided Media
〰️

Analog vs Digital Signals

Analog signals are continuous waves that vary smoothly over time (like sound waves). Digital signals are discrete, represented as 0s and 1s. Modern systems primarily use digital signals for greater noise resistance and reliability.

Key terms: bandwidth, frequency, amplitude, period, wavelength, bit rate, baud rate.

🔌

Twisted Pair Cable

Two copper wires twisted together to reduce electromagnetic interference (EMI). Used widely in telephone systems and Ethernet (Cat 5, Cat 6). Available as UTP (unshielded) and STP (shielded).

Range: up to 100m. Speed: up to 10 Gbps (Cat 6a).

📡

Coaxial Cable

A central conductor surrounded by insulation, a metallic shield, and an outer jacket. Higher bandwidth than twisted pair. Used in cable TV and early Ethernet (10Base2). Excellent noise immunity.

Two types: Thinnet (10Base2) and Thicknet (10Base5).

💡

Fiber Optic Cable

Transmits data as pulses of light through glass or plastic fibers. Extremely high bandwidth, immune to electromagnetic interference, and suitable for long distances. Powers the internet backbone.

Types: Single-mode (long distance) and Multi-mode (short range).

📶

Radio Waves

Omnidirectional unguided waves used in AM/FM radio, WiFi (802.11), Bluetooth, and cellular networks. Penetrate walls and travel long distances, but susceptible to interference and security risks.

Frequency range: 3 Hz – 3 GHz.

🛰️

Microwaves & Satellites

Directional high-frequency waves used for point-to-point communication, satellite links, and cellular backhaul. Satellites in geostationary (GEO), medium-earth (MEO), and low-earth (LEO) orbits enable global coverage.

Key concept: propagation delay increases with orbital altitude.

Protocols &
Networking Models

The OSI Model (7 Layers)

The Open Systems Interconnection model standardises how different computer systems communicate. Each layer serves the one above it and is served by the one below it.

# Layer Function Protocol
7ApplicationUser interface, servicesHTTP, FTP, DNS
6PresentationData format, encryptionSSL, JPEG
5SessionSession managementNetBIOS
4TransportEnd-to-end deliveryTCP, UDP
3NetworkRouting & addressingIP, ICMP
2Data LinkFraming, error detectionEthernet, MAC
1PhysicalBits on the mediumCable, Radio

Mnemonic: "All People Seem To Need Data Processing"

TCP/IP Model

The practical implementation used by the internet. It has four layers, condensing the OSI model:

Application Layer

HTTP, FTP, SMTP, DNS — services users interact with.

Transport Layer

TCP (reliable, connection-oriented) vs UDP (fast, connectionless).

Internet Layer

IP addressing and routing between networks.

Network Access Layer

Physical transmission — Ethernet, WiFi, and link control.

Error Detection Methods

Reliable data communication requires detecting and correcting errors caused by noise or interference:

  • Parity bit — adds a single bit to make 1s even or odd
  • Checksum — sums all data values; receiver verifies total
  • CRC — Cyclic Redundancy Check; most powerful and widely used
  • Hamming code — detects AND corrects single-bit errors

Common Protocols Quick Reference

HTTP / HTTPS
Web browsing; HTTPS adds TLS encryption. Port 80 / 443.
FTP
File Transfer Protocol. Transfers files between client and server. Port 21.
SMTP
Simple Mail Transfer Protocol. Sends email between servers. Port 25.
DNS
Domain Name System. Translates domain names to IP addresses. Port 53.
TCP
Transmission Control Protocol. Reliable, ordered, connection-oriented delivery.
UDP
User Datagram Protocol. Fast, connectionless; used in video streaming, gaming.
IP (v4/v6)
Internet Protocol. Logical addressing and packet routing across networks.
DHCP
Dynamic Host Configuration Protocol. Assigns IP addresses automatically. Port 67/68.
